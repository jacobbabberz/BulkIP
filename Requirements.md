# Purpose
We need a bulk IP lookup web tool
We like the format that 'https://app.spur.us/search?q=8.8.8.8' gives us, an example can be seen below
{
  "as": {
    "number": 15169,
    "organization": "Google LLC"
  },
  "infrastructure": "DATACENTER",
  "ip": "8.8.8.8",
  "location": {
    "city": "Anycast",
    "country": "ZZ",
    "state": "Anycast"
  },
  "organization": "Google LLC"
}
this needs to be able to extract ipv4 and ipv6 ips when a log is pasted in, and ignore the rest of the log
we'd like this to utilize free/open source lookups like whois
